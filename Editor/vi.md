# vi editor

## Super-Basic usage you would like to know
- :(colon) : change mode, search string or regex pattern
- _n_ dd (or dd _n_) : delete _n_ lines
- ^ : Beginning of line
- $ : End of line
- Ctrl + [ : Goto prev "paragraph"
- Ctrl + ] : Goto next "paragraph"
- /word/ : Goto line contains "word"
